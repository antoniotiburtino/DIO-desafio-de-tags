  • tag h1 até h6

    Começando pela tag h1, ela é mais usada pra fazer o titulo principal do site.( o nosso h1 está em DESAFIOS SOBRE AS TAGS). assim como a tag h1, temos outras tags como h2,h3,h4,h5e h6, ou seja, vai do h1 até o h6. Onde h1 é o maior e h6 o menor.

  • Tag p

    A tag "p" é usada para criamos paragrafos

  • tag mark

    A tag mark é usada para marcamos um texto, como se fosse um marca texto mesmo.

  • Tag small

    A tag small é para adicionarmos frase, numero ou outra coisa menor.

  • tag i

    A tag i é para deixae o texto em italico .

  • Tag u

    A tag deixa o texto sublinhado

  • Tag strong

    A tag strong deixa em negrito

  • Tag ul

    A tag ul é referente a lista não ordenada, esta vendo essas bolinhas pretas? elas sao referentes a ol

(Assim como ul e ol, podemos usar os "li" dentro das listas)

  • Tag li

    A tag li é para representar um item dentro das listas.

  • Tag a

    A tag a é para criarmos um link, seja para ir para outro site, ou ir para outro lugar do site atual.

  • Tag hr

    A tag hr representa uma quebra temática entre elementos de nível de parágrafo. Usamos ela para separar o nosso titulo principal.

  • Tag sup

    A tag sup é um trecho do texto pode ser formatado como subscrito ou sobrescrito usando-se as tags sub ou sup. Exemplo: teste teste

  • Tag sub

    A tag sub é o efeito contrario da tag sup. Exemplo: teste teste

  • Tag blockquote

    A tag blockquote indica que o texto dentro dela é uma citação longa. Exemplo:

    teste teste

A tag font ela foi descontinuada no HTML 5

  • Tag del

    A tag del representa uma parte do texto que foi excluída de um documento. Exemplo: minha cor favorita é blue

  • Tag abbr

    A tag abbr define uma abreviação ou um acrônimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".
